# 框架简介

**LangChain** 是一个用于开发大语言模型（LLM）驱动应用程序的框架。

LangChain 简化了大语言模型应用生命周期的每个阶段：
- **开发阶段**：使用 LangChain 开源[组件](/docs/concepts)和[第三方集成](/docs/integrations/providers/)构建应用
通过 [LangGraph](/docs/concepts/architecture/#langgraph) 构建具备一流流式支持与人工介入能力的有状态智能体
- **产品化阶段**：使用 [LangSmith](https://docs.smith.langchain.com/) 进行应用检查、监控与评估，实现持续优化和可靠部署
- **部署阶段**：通过 [LangGraph Platform](https://langchain-ai.github.io/langgraph/cloud/) 将 LangGraph 应用转化为生产级 API 和智能助手

## 架构体系

LangChain 框架包含多个开源库，详见[架构](/docs/concepts/architecture/)页面：

- **`langchain-core`**：聊天模型和其他组件的核心抽象层
- **集成包**（如 `langchain-openai`, `langchain-anthropic` 等）：重要集成已拆分为轻量级包，由 LangChain 团队与集成开发者共同维护
- **`langchain`**：构成应用认知架构的链式结构、智能体和检索策略
- **`langchain-community`**：社区维护的第三方集成
- **`langgraph`**：将 LangChain 组件组合成具备持久化、流式处理等关键特性的生产级应用的编排框架，详见 [LangGraph 文档](https://langchain-ai.github.io/langgraph/)
